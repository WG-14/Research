## Design priority

Prioritize lifecycle safety over cosmetic changes.

Do not:

* add another random branch inside `paper_execute()`
* mark partial fills as terminal `FILLED`
* release dedup for `PARTIAL`, `SUBMIT_UNKNOWN`, `ACCOUNTING_PENDING`, or `RECOVERY_REQUIRED`
* silently fall back from invalid stress config to immediate mode
* weaken existing lot/dust, guardrail, fee, dedup, or accounting behavior
* make risky live order behavior changes

The patch must preserve the core direction: paper runtime should rehearse the same failure families live OMS must survive.

## Required patch 1: Add unresolved-order gate to paper execution

Add a paper unresolved-order gate before creating/claiming a new order intent.

The intended behavior:

```text
If any unresolved/open order exists locally:
  PENDING_SUBMIT
  NEW
  PARTIAL
  SUBMIT_UNKNOWN
  ACCOUNTING_PENDING
  RECOVERY_REQUIRED
  CANCEL_REQUESTED

Then paper_execute() must not create a new paper order intent.
It must return None.
It must emit durable/loggable evidence explaining the skip.
```

Use the existing OMS helper if appropriate:

```python
evaluate_unresolved_order_gate(
    conn,
    now_ms=...,
    max_open_order_age_sec=int(settings.MAX_OPEN_ORDER_AGE_SEC),
)
```

Prefer placing this after `init_portfolio(conn)` and before sizing/intent-claiming/order creation. Preserve default immediate behavior; ordinary immediate paper orders should still terminalize as `FILLED` and therefore not block the next order unless an open/unresolved order actually remains.

Expected evidence:

* At least a run log entry using repo style, for example `format_log_kv`.
* Prefer also recording a durable `order_events` row if there is an existing suitable helper and no unsafe fake order id is required.
* A notification is acceptable if consistent with existing paper skip behavior.
* Do not create a second order as a way to record the block.

If durable event recording for a skip without a new order is awkward, document the tradeoff and at minimum add a test using `caplog` or DB state to prove the gate fired.
